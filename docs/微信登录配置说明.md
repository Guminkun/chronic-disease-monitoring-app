# 微信登录配置说明

## 功能概述

本项目已集成微信小程序登录功能，用户可以通过微信授权快速登录，无需输入手机号和密码。

## 后端配置

### 1. 环境变量配置

在 `backend/.env` 文件中添加以下配置：

```env
# 微信小程序配置
WECHAT_APPID=你的微信小程序AppID
WECHAT_SECRET=你的微信小程序AppSecret
```

### 2. 获取微信小程序凭证

1. 访问 [微信公众平台](https://mp.weixin.qq.com/)
2. 登录后进入「开发」->「开发管理」->「开发设置」
3. 复制 AppID 和 AppSecret

### 3. 数据库迁移

运行迁移脚本添加微信相关字段：

```bash
cd backend
python add_wechat_fields.py
```

该脚本会在 `users` 表中添加以下字段：
- `wechat_openid`: 微信用户唯一标识
- `wechat_unionid`: 微信开放平台唯一标识（可选）
- `wechat_nickname`: 微信昵称
- `wechat_avatar`: 微信头像

## 前端配置

### 1. 微信小程序配置

在微信小程序管理后台配置服务器域名：

1. 登录 [微信公众平台](https://mp.weixin.qq.com/)
2. 进入「开发」->「开发管理」->「开发设置」->「服务器域名」
3. 添加后端API域名到 `request合法域名`

### 2. manifest.json 配置

确保 `frontend-mobile/src/manifest.json` 中已配置微信小程序 AppID：

```json
{
  "mp-weixin": {
    "appid": "你的微信小程序AppID"
  }
}
```

## API 接口

### 微信登录接口

**POST** `/auth/wechat/login`

**请求参数：**
```json
{
  "code": "微信登录凭证"
}
```

**响应：**
```json
{
  "access_token": "JWT令牌",
  "token_type": "bearer",
  "role": "patient",
  "user_id": "用户ID"
}
```

## 登录流程

1. 用户在登录页面点击「微信登录」
2. 前端调用 `uni.login()` 获取微信登录凭证 `code`
3. 前端将 `code` 发送到后端 `/auth/wechat/login` 接口
4. 后端使用 `code` 向微信服务器换取 `openid` 和 `session_key`
5. 后端根据 `openid` 查找或创建用户
6. 后端生成 JWT token 返回给前端
7. 前端保存 token 并跳转到主页

## 测试说明

### 开发环境测试

在微信开发者工具中测试时，需要注意：

1. 确保已配置正确的 AppID
2. 在开发者工具中勾选「不校验合法域名」选项
3. 使用真机调试时需要配置合法域名

### 模拟测试

如果暂时没有微信小程序凭证，可以：

1. 使用密码登录功能
2. 或在后端添加测试用的 mock 接口

## 注意事项

1. **安全性**：AppSecret 必须保密，切勿提交到代码仓库
2. **用户隐私**：首次微信登录会自动创建用户账号
3. **账号绑定**：后续可以开发手机号绑定功能，将微信账号与手机号关联
4. **UnionID**：如需在多个小程序/公众号间共享用户，需要配置 UnionID

## 常见问题

### Q: 登录失败提示 "WeChat login not configured"

A: 检查 `.env` 文件中是否正确配置了 `WECHAT_APPID` 和 `WECHAT_SECRET`

### Q: 登录失败提示 "WeChat login failed"

A: 检查 AppID 和 AppSecret 是否正确，以及网络是否正常

### Q: 真机调试时请求失败

A: 确保已在微信公众平台配置了服务器域名白名单
