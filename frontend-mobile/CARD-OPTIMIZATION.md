# 卡片视觉层次优化方案

## 问题分析

原有的背景色系统存在视觉层次不足的问题：

**原方案**：
- 页面背景：`#F5F7FA` (浅灰)
- 卡片背景：`#FFFFFF` (纯白)
- 对比度：较弱，卡片不够突出

**视觉问题**：
- 卡片和背景色对比度不够明显
- 视觉层次扁平，缺乏深度感
- 用户难以快速识别可交互区域

## 优化方案

### 1. 背景色调整

**新方案**：
```scss
--color-bg-base: #E8ECF1;        /* 页面背景 - 更深的灰色 */
--color-bg-elevated: #FFFFFF;    /* 提升层 - 保持纯白 */
--color-bg-muted: #F1F5F9;       /* 次要背景 - 调整对比度 */
--color-bg-subtle: #EFF6FF;      /* 柔和背景 - 主题色 */
```

**对比度提升**：
- 背景色从 `#F5F7FA` 调整为 `#E8ECF1`
- 明度降低约 5%，对比度提升约 15%
- 卡片白色更加突出，层次分明

### 2. 卡片样式增强

**新增卡片专用变量**：
```scss
--color-card-bg: #FFFFFF;
--color-card-border: rgba(148, 163, 184, 0.1);
--shadow-card: 0 4rpx 20rpx rgba(0, 0, 0, 0.08), 
              0 0 1rpx rgba(0, 0, 0, 0.05);
```

**设计特点**：
- ✅ 纯白色背景，最大化对比度
- ✅ 双层阴影，增强立体感
- ✅ 微妙边框，增强边界感
- ✅ 按压反馈，提升交互体验

### 3. 阴影系统优化

**原阴影**：
```scss
--shadow-sm: 0 4rpx 16rpx rgba(0, 0, 0, 0.04);  /* 过轻 */
--shadow-md: 0 8rpx 24rpx rgba(0, 0, 0, 0.06);  /* 过轻 */
```

**新阴影**：
```scss
--shadow-sm: 0 4rpx 16rpx rgba(0, 0, 0, 0.06);  /* 提升 */
--shadow-md: 0 8rpx 24rpx rgba(0, 0, 0, 0.10);  /* 提升 */
--shadow-card: 0 4rpx 20rpx rgba(0, 0, 0, 0.08), 
              0 0 1rpx rgba(0, 0, 0, 0.05);     /* 双层 */
```

**优化效果**：
- 阴影透明度提升 50%-67%
- 新增双层阴影，增强深度感
- 卡片立体感更强，视觉层次更明显

## 设计原则

### 1. 视觉层次清晰
- 使用不同明度的灰色区分层次
- 白色卡片最大程度突出显示
- 阴影增强立体感

### 2. 一致性
- 所有卡片使用统一的设计令牌
- 统一的圆角、阴影、边框
- 统一的交互动画

### 3. 可访问性
- 背景与卡片对比度 > 1.5:1
- 文字与背景对比度 > 4.5:1
- 支持按压反馈动画

### 4. 性能优化
- 使用 CSS 变量，便于主题切换
- 避免过度阴影，保证性能
- 合理使用边框替代阴影

## 使用指南

### 标准卡片
```scss
.card {
  background: var(--color-card-bg);
  border-radius: var(--radius-lg);
  box-shadow: var(--shadow-card);
  border: 1rpx solid var(--color-card-border);
}
```

### 强调卡片
```scss
.card-highlight {
  background: var(--color-card-bg);
  border-radius: var(--radius-lg);
  box-shadow: var(--shadow-md);  /* 更强的阴影 */
  border: 2rpx solid var(--color-primary-light);
}
```

### 禁用状态
```scss
.card-disabled {
  background: var(--color-bg-muted);
  border-radius: var(--radius-lg);
  box-shadow: none;
  border: 1rpx solid rgba(148, 163, 184, 0.2);
  opacity: 0.6;
}
```
