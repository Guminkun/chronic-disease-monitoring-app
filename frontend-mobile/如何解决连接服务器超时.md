# 小程序连接服务器超时问题解决方案

## 错误信息
```
Uncaught (in promise) Error: Could not establish connection. Receiving end does not exist.
```

## 原因分析
1. 后端服务未启动
2. 小程序API地址配置错误
3. 微信开发者工具网络设置问题
4. CORS跨域配置问题

## 解决步骤

### 1. 启动后端服务

在 `backend` 目录下运行：
```bash
cd E:\AI\baiduyun\chronic-disease-monitoring-app-0330\backend
python -m uvicorn app.main:app --reload --host 0.0.0.0 --port 8000
```

验证服务是否启动：
- 浏览器访问：http://localhost:8000/docs
- 应该能看到 FastAPI 自动文档页面

### 2. 检查微信开发者工具设置

**重要：必须开启以下设置**

1. 打开微信开发者工具
2. 点击右上角"详情"
3. 选择"本地设置"标签
4. **勾选以下选项**：
   - ✅ 不校验合法域名、web-view（业务域名）、TLS版本以及HTTPS证书
   - ✅ 不校验安全域名、TLS版本以及HTTPS证书
   - ✅ 启用调试

### 3. 检查小程序API配置

文件：`frontend-mobile/src/utils/request.ts`

确认配置：
```typescript
// 小程序本地开发直连后端
const MP_BASE_URL = 'http://127.0.0.1:8000'  // 或 'http://localhost:8000'
```

### 4. 验证网络连接

在微信开发者工具控制台运行：
```javascript
// 测试后端连接
uni.request({
  url: 'http://127.0.0.1:8000/docs',
  success: (res) => {
    console.log('连接成功', res)
  },
  fail: (err) => {
    console.error('连接失败', err)
  }
})
```

### 5. 检查防火墙

确保防火墙允许 8000 端口：
```bash
# Windows防火墙添加规则（管理员权限）
netsh advfirewall firewall add rule name="FastAPI" dir=in action=allow protocol=TCP localport=8000
```

### 6. 重启服务顺序

按以下顺序重启：

1. 关闭微信开发者工具
2. 启动后端服务
3. 等待后端启动完成（看到 "Uvicorn running on http://0.0.0.0:8000"）
4. 打开微信开发者工具
5. 刷新小程序

## 常见问题

### Q1: 后端服务启动失败
检查 8000 端口是否被占用：
```bash
netstat -ano | findstr :8000
```

如果被占用，杀死进程或更换端口。

### Q2: 真机调试无法连接
真机调试需要：
1. 手机和电脑在同一局域网
2. 后端服务监听 `0.0.0.0`（不是 `127.0.0.1`）
3. 使用电脑的局域网IP地址（如 `http://192.168.1.100:8000`）

查看本机IP：
```bash
ipconfig | findstr "IPv4"
```

### Q3: 请求超时
增加请求超时时间：
```typescript
// frontend-mobile/src/utils/request.ts
uni.request({
  timeout: 30000, // 30秒
  // ...
})
```

### 方案1：开发环境（推荐）

1. 启动后端：
```bash
cd E:\AI\baiduyun\chronic-disease-monitoring-app-0330\backend
python -m uvicorn app.main:app --reload --host 0.0.0.0 --port 8000
```

2. 启动前端编译：
```bash
cd E:\AI\baiduyun\chronic-disease-monitoring-app-0330\frontend-mobile
npm run dev:mp-weixin
```

3. 打开微信开发者工具，导入项目：
```
E:\AI\baiduyun\chronic-disease-monitoring-app-0330\frontend-mobile\dist\dev\mp-weixin
```

### 方案2：使用启动脚本

创建 `启动开发环境.bat`：
```batch
@echo off
start cmd /k "cd /d E:\AI\baiduyun\chronic-disease-monitoring-app-0330\backend && python -m uvicorn app.main:app --reload --host 0.0.0.0 --port 8000"
timeout /t 5
start cmd /k "cd /d E:\AI\baiduyun\chronic-disease-monitoring-app-0330\frontend-mobile && npm run dev:mp-weixin"
echo 请等待编译完成后，打开微信开发者工具
pause
```

## 仍然无法解决？

1. 检查后端日志是否有错误
2. 检查微信开发者工具控制台是否有其他错误
3. 尝试清除缓存：微信开发者工具 → 清缓存 → 全部清除
4. 重启电脑后再试
