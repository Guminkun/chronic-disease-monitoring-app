## 保留的功能

### navigate.ts 工具函数

虽然恢复了原始头部样式，但保留了 `utils/navigate.ts` 工具函数，用于：

1. 解决微信小程序导航超时问题
2. 自动处理页面栈溢出
3. 提供重试机制

你可以在任何页面使用：

```typescript
import { safeNavigate, safeNavigateBack } from '@/utils/navigate'

// 安全跳转
safeNavigate({ url: '/pages/xxx/xxx' })

// 安全返回
safeNavigateBack()
```
